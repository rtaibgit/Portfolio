# My Portfolio

This repo is where I put some of the projects I've been building while learning programming, cloud, and DevOps. I really enjoy learning new technical skills and pushing myself to understand how things actually work behind the scenes, so I like building small projects that let me experiment and apply what I'm learning.

## Projects

### CI/CD Pipeline (AWS + GitHub Actions)

This project sets up a simple CI/CD pipeline that automatically deploys a website to an AWS S3 bucket.

Whenever I push changes to the `main` branch, GitHub Actions runs a workflow that uploads the latest version of the site to S3. The bucket is configured for static website hosting, so the site updates automatically whenever new code is pushed.

Working on this project helped me understand how automated deployment pipelines work and how cloud services like AWS can be used to host and update applications.

Tech used:

* AWS S3
* GitHub Actions
* YAML
* HTML / CSS

### Python Café Project

This is a small Python program that simulates a café ordering system.

The program allows users to choose items from a menu and calculates the total cost of the order. While it's a simple project, it helped me practise writing structured Python code, working with user input, and thinking through program logic step by step.

I enjoy projects like this because they help me build confidence with programming while solving small real-world style problems.

Tech used:

* Python
* program logic
* user input handling
